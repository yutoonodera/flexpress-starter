# FlexPress Starter（日本語）

> 更新頻度の高い企業向けのWebサイト構築スターターキット

FlexPress Starter は、Next.js と Headless WordPress を組み合わせた  
企業向けWebサイトのためのスターターキットです。

---

## 特徴

- Next.js（App Router）
- Headless WordPress
- SEO最適化
- 本番運用を前提とした構成

---

## コンテンツ出し分け

FlexPress Starter では、サイト内の任意の場所に  
条件付きの記事一覧を表示できます。

カテゴリ・店舗・エリアなどの条件によって、  
ページごとに最適なコンテンツを出し分けることが可能です。

### 利用例

- トップページに「お知らせ」だけを表示
- 店舗ページに「店舗別の記事」を表示
- エリアページに「地域ごとの記事」を表示
- 事業ページに「カテゴリ別ブログ」を表示

### できること

- カテゴリごとの記事一覧表示
- タグ・カスタムタクソノミーでの絞り込み
- 店舗・エリアごとのコンテンツ出し分け
- 任意のページへの記事一覧埋め込み
