# FlexPress Starter（日本語）

> 更新頻度の高い企業向けのWebサイト構築スターターキット

FlexPress Starter は、Next.js と Headless WordPress を組み合わせた  
企業向けWebサイトのためのスターターキットです。

---

## 特徴

- Next.js（App Router）
- Headless WordPress
- SEO最適化
- 本番運用を前提とした構成

---

## 想定業種

FlexPress Starter は、更新頻度の高い企業サイトに最適化されています。

### 🏢 不動産仲介業
- 物件情報の更新
- エリア別コンテンツ
- お知らせ管理

### 🏬 多店舗チェーン
- 店舗一覧
- キャンペーン情報
- 地域別ページ

### 📰 メディアサイト
- 記事投稿
- カテゴリ管理
- SEOを意識した構成

---

### 必要環境
- Docker

docker compose -f docker-compose.local.yml up -d
