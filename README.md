# FlexPress Starter

A Next.js + Headless WordPress starter kit for content-heavy business websites.

## Features
- Next.js (App Router)
- Headless WordPress
- SEO optimized
- Production ready

## Getting Started

```bash
docker compose -f docker-compose.local.yml up
```
